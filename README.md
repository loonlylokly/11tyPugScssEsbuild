## Install

1. Make a directory and navigate to it:

```
mkdir new-site
cd new-site
```

2. Clone this Repository

```
git clone https://github.com/loonlylokly/11tyPugScssEsbuild.git .
```

3. Install dependencies

```bash
npm install
```

4. Run ptoject

Generate a production-ready build to the `_dist` folder:
```bash
npm run dev
```
